# Samsung Global Sales Analytics (2021–2024)

## 📋 Project Overview
An end-to-end data analytics project focused on analyzing Samsung's global sales performance from 2021 to 2024.

The dataset contains 36,000+ rows of data across 52 countries. The objective of this project is to identify revenue drivers, market trends, and operational performance using SQL, Excel, and Power BI.

---

## 🛠️ Tech Stack

- **SQL** – Data extraction, multi-table joins, and data cleaning  
- **Excel** – Initial exploratory data analysis (EDA) and data validation  
- **Power BI** – Data modeling (Star Schema), DAX measures, and interactive dashboard development  

---

## 🏗️ Data Architecture

The project is built using a **Star Schema** in Power BI to improve performance and scalability.

**Fact Table:**
- Fact_Sales (Transactions, Revenue, Units, Discounts)

**Dimension Tables:**
- Dim_Product  
- Dim_Geography  
- Dim_Date  
- Dim_Channel  

---

## 🚀 Key Insights

- **Total Revenue:** $22.91M generated between 2021–2024  
- **Top Product Category:** Smart TV is the primary revenue driver  
- **Top Region:** Europe generates the highest revenue share  
- **5G Adoption:** 32.7% contribution, indicating strong future growth potential  
- **Return Rate:** 7.0% globally, highlighting areas for operational improvement  

---

## 📊 Dashboard Preview

The Power BI dashboard includes:

- Executive Summary (High-level KPIs and revenue trends)  
- Product Deep-Dive (Sales performance and tech adoption analysis)
